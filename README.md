# discord-spotify-bot

This project allows users to explore and share music via the Spotify API through the discord chat app. The Spotify API commands used for the project do not require OAuth.
___
## Tools used
* [discord.py(docs)](http://discordpy.readthedocs.io/en/latest/api.html)
* [Spotify.API](https://developer.spotify.com/web-api/console/)
